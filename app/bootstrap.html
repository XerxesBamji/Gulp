Installing Bootstrap, jQuery & Popper.js with npm

1. Installing Bootstrap, jQuery & Popper.js

The first thing you need to do is to look at your package.json file. Under your devDependencies, look to see if you have any Bootstrap, jQuery or popper.js installed. If not, add them to the devDependencies. Your code will then look like this: 

{
  "name": "gulp-test",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "author": "",
  "license": "ISC",
  "devDependencies": {
    "gulp-google-webfonts": "^2.1.0",
    "bootstrap": "^4.3.1",
    "jquery": "^3.4.0",
    "popper.js": "^1.15.0",
    "browser-sync": "^2.26.3",
    "gulp": "^4.0.0",
    "gulp-autoprefixer": "^6.0.0",
    "gulp-cache": "^1.1.1",
    "gulp-concat": "^2.6.1",
    "gulp-imagemin": "^5.0.3",
    "gulp-rename": "^1.4.0",
    "gulp-sass": "^4.0.2",
    "gulp-uglify": "^3.0.2"
  }
}

Once you have added this into your package.json. Deleted your package-lock.json. Then in your working directory, run the command:

npm install

Once this is finished. You will notice in node_modules that Bootstrap, jQuery and popper.js have been added and can now be used.

2. Linking Bootstrap

To link bootstrap. You will need to import the bootstrap.scss file into your styles.scss. Add this code into your styles.scss:

@import "../../node_modules/bootstrap/scss/bootstrap";

Once this has been added, run gulp sass and Bootstrap sass will be added to your css.

npm install

Once this is finished. You will notice in node_modules that Bootstrap, jQuery and popper.js have been added and can now be used.

3. Linking jQuery & popper.js

As bootstrap.bundle includes bootstrap and popper.js, we only need two links. Bootstrap.bundle and jQuery. The links would look as follows:

<script src="../node_modules/jquery/dist/jquery.min.js"></script> <script src="../node_modules/bootstrap/dist/js/bootstrap.bundle.js"></script>

Once this has been added, run gulp sass and Bootstrap sass will be added to your css.

npm install

Once this is finished. You will notice in node_modules that Bootstrap, jQuery and popper.js have been added and can now be used.
